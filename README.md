# Math Whiteboard Demo

Демо-витрина функциональности интерактивной рабочей тетради (whiteboard) для математики.

## Скриншот
![Рабочая тетрадь — главная страница](./docs/screenshots/whiteboard-workbook.png)

## Что показано
- Инструменты 2D/3D построений.
- Работа с сечениями 3D-фигур.
- Трансформации объектов на доске.
- Режимы выделения, работы с линиями, точками, подписями.
- Сохранение состояния сессии.

## Стек
- React + TypeScript + Vite
- MUI
- mock API/server

## Быстрый старт
```bash
npm install
npm run dev:showcase
```

Откроется маршрут: `/workbook` (если браузер не открылся автоматически: [http://localhost:5173/workbook](http://localhost:5173/workbook)).

## Демо-доступ
- Учитель: `teacher@axiom.demo` / `magic`
- Студент: `student@axiom.demo` / `magic`

## Что оценивать на собеседовании
- Архитектура редактора доски.
- Модель данных объектов и событий.
- UX интерактивных инструментов.
- Подход к realtime и автосохранению.

## Ограничения
- В демо используется mock backend.
- Функции внешних AI/медиа-провайдеров отключены.
